### Design Elements

#### Grid System
- Base unit: 8px
- Margin: 24px
- Gutter: 16px
- Column count: 12
- Container max-width: 1200px

#### Box Highlights
- Background: Primary colors at 10% opacity
- Border: 1px solid primary color
- Glow effect: 0 0 20px rgba(primary-color, 0.3)
- Corner radius: 4px

#### Glitch Effect
- RGB split distance: 2-4px
- Animation duration: 0.2s
- Trigger: Hover/Focus states
- Use sparingly on key elements only
